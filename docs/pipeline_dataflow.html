相关性验证内部子流程

逐试验 × 逐轴 (x / z)
输入 A
视频 CoM 序列
真实帧 · ≈30 Hz
输入 B
Vicon CoM 序列
真实帧 · 250 Hz
Step 1
建立时间轴
video_t = (frame − 1) / 29.996 ｜ vicon_t = (frame − first_frame) / 250（两者 t=0 已同步）
Step 2
截取时间重叠窗口
t_min = max(起点) ~ t_max = min(终点)，对视频序列取 mask；裁掉 >20% 则记 warning
Step 3 ★
np.interp 线性重采样
方向：Vicon → 视频时间网格。以每个视频时间点为查询点，在相邻两个 Vicon 样本间按时间比例插值
np.interp(video_t, vicon_t, vicon_vals) → 输出长度 = 视频帧数（250 Hz 下采样到 ~30 Hz）
有效样本量 = 视频帧数（不凭空增点，避免伪重复抬高相关）
Step 4
符号对齐
vicon_Y → −vicon_Y（左右），vicon_Z → −vicon_Z（垂直：像素 Y 向下 vs Vicon Z 向上）
Step 5
线性去趋势 + Z-score 归一化
scipy.signal.detrend 去除步行整体漂移 → (x−μ)/σ 消除量纲（像素 vs 毫米），只比形状节律
Pearson r + p 值
直接对齐相关
nRMSE
归一化均方根误差
xcorr peak_r / lag
最优时移 + 最大相关
